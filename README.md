# 🐱 CatPhotoApp

A simple responsive web page for cat lovers, featuring interactive elements such as buttons, checkboxes, radio buttons, and a submission form. Styled using Google Fonts, Font Awesome icons, and Bootstrap grid classes.

## 🚀 Features

- 📷 Display of relaxing and running cat photos
- 🖼 Thick green border around the main cat image
- 🎨 Custom font using [Google Fonts: Lobster](https://fonts.google.com/specimen/Lobster)
- 💠 Font Awesome icons for buttons
- ✅ Interactive form with:
  - Radio buttons for indoor/outdoor preference
  - Checkboxes for cat personality traits
  - Text input for cat photo URL submission
- 🧩 Responsive layout using Bootstrap's grid system

## 🛠 Technologies Used

- HTML5
- CSS3
- Bootstrap (Grid System Only)
- [Google Fonts](https://fonts.google.com/)
- [Font Awesome 5.8.1](https://fontawesome.com/)

## 📁 File Structure

``

index.html

``

_All code is self-contained within a single HTML file._

## 🧪 How to Use

1. Clone or download this repository.
2. Open the `index.html` file in any modern browser.
3. Interact with buttons, checkboxes, and the form to explore features.
